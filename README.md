# Employee Management System

## Project Overview
The Employee Management System is a full-stack web application built using React for the frontend and Spring Boot for the backend.
This application allows users to manage employee records efficiently by performing CRUD (Create, Read, Update, Delete) operations.

## Features
. Add new employees
. View all employees
. Update employee details
. Delete employee records
. Search employees
. RESTful API integration
. Responsive user interface

## Architecture
React handles the user interface and communicates with backend APIs.
Spring Boot provides REST APIs.
MySQL stores employee data.
Git & GitHub are used for version control.

## How to Run the Project
# Backend (Spring Boot):
1.Open backend project in VS Code
2.Configure MySQL database in application.properties
3.Run the Spring Boot application
4.Backend runs at:
http://localhost:9191

# Frontend (React):
1.Navigate to frontend folder
2.Install dependencies: npm install
3.Start React app: npm start
4.Frontend runs at:
http://localhost:3000
